📘 Retrieval-Augmented Generation (RAG) Document QA System

This project showcases how to build a Retrieval-Augmented Generation (RAG) pipeline that combines LLMs with external documents. It loads PDFs/text, generates embeddings, stores them in a vector database, and retrieves the most relevant chunks to provide context-aware, accurate answers.

🔹 Features

Load and process PDF/Text documents

Create embeddings & store in vector DB (Chroma/FAISS)

Query documents using LLMs with augmented context

Easy-to-follow Jupyter Notebook tutorials

🔹 Tech Stack

Python · LangChain · PyMuPDF/PyPDF · FAISS/Chroma · OpenAI/Hugging Face
